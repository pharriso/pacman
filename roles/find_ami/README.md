find_ami
=========

finds AMI ID in ec2 region

Role Variables
--------------

The following variables can be overriden.

| Name                    | Default value         |                                                                                  |
|-------------------------|-----------------------|----------------------------------------------------------------------------------|
| ec2_region              |                       | EC2 region                                                                       |
| instance_info           |                       | AMI instance info                                                                |

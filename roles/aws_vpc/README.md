aws_vpc
=========

creates ec2 networking

Role Variables
--------------

The following variables can be overriden.

| Name                    | Default value         |                                                                                  |
|-------------------------|-----------------------|----------------------------------------------------------------------------------|
| ec2_region              |                       | EC2 region                                                                       |
| vpc_name                | rhel8                 | os type                                                                          |
| vpc_cidr                |                       | SSH Keypair name                                                                 |
| security_group          |                       | AMI ID                                                                           |
| security_group_rules    |                       | Secuirty group name                                                              |
